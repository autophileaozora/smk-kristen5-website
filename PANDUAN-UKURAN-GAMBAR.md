# Panduan Ukuran Gambar - Website SMK Kristen 5 Klaten

Dokumen ini mencantumkan ukuran/dimensi gambar yang digunakan pada setiap halaman website,
agar desainer dapat menyiapkan aset gambar dengan ukuran yang tepat.

---

## Ketentuan Umum

- Format yang didukung: **JPG, PNG, WebP, GIF**
- Ukuran file maksimal: **2 MB per gambar**
- Semua gambar dikompresi dan di-resize otomatis oleh **Cloudinary** setelah diupload

### Batas resize otomatis Cloudinary

| Kategori     | Maks. Lebar | Berlaku untuk                                      |
|--------------|-------------|----------------------------------------------------|
| Large        | 1200 px     | Hero slider, fasilitas, jurusan, galeri, CTA       |
| Medium       | 800 px      | Artikel, ekskul, prestasi                          |
| Thumbnail    | 400 px      | Logo partner/mitra, foto alumni, foto profil       |

> Siapkan file dalam resolusi asli yang lebih besar dari batas di atas agar hasil kompresi tetap tajam.

## Homepage

### Hero Slider (background slide)

| Properti       | Nilai                          |
|----------------|-------------------------------|
| Lebar tampil   | Full width layar (100vw)      |
| Tinggi tampil  | Min 700px (desktop), min 600px (tablet), min 88vh (mobile) |
| Rasio ideal    | **16:9**                      |
| Rekomendasi    | **1920 x 1080 px**            |
| Catatan        | Bagian tengah gambar yang paling terlihat. Hindari teks/elemen penting di tepi. |

### Program / Jurusan

| Posisi              | Tinggi tampil | Rasio ideal | Rekomendasi file |
|---------------------|--------------|-------------|-----------------|
| Desktop (panel kiri) | 420 px       | 4:3         | **1200 x 900 px** |
| Mobile (dalam accordion) | 192 px  | 16:5        | **800 x 300 px**  |

### Kegiatan / Ekstrakurikuler (carousel section)

| Properti      | Nilai                                              |
|---------------|----------------------------------------------------|
| Tinggi tampil | 250px (mobile) - 350px (tablet) - 450px (desktop) |
| Lebar tampil  | Full width                                         |
| Rasio ideal   | **16:9**                                           |
| Rekomendasi   | **1200 x 675 px**                                  |

### Artikel (section berita)

| Posisi                 | Tinggi tampil               | Rekomendasi file  |
|------------------------|-----------------------------|-------------------|
| Artikel utama (besar)  | 220px (mobile) - 320px (desktop) | **1200 x 500 px** |
| Artikel kecil (thumbnail grid) | ~120px                | **800 x 450 px**  |

### Testimoni / Alumni

| Properti    | Nilai                          |
|-------------|-------------------------------|
| Tampil      | Foto bulat (circle), 40-48 px |
| Rasio       | **1:1 (persegi)**             |
| Rekomendasi | **200 x 200 px**              |
| Catatan     | Pastikan wajah di tengah gambar |

### Partner / Mitra (logo)

| Properti    | Nilai                                        |
|-------------|---------------------------------------------|
| Tampil      | ~160px lebar, tinggi otomatis               |
| Rasio       | Bebas (landscape disarankan)                |
| Rekomendasi | **400 x 200 px**, PNG dengan latar transparan |
| Catatan     | Gunakan format PNG transparan untuk hasil terbaik |

### CTA Section (background)

| Properti      | Nilai                                    |
|---------------|------------------------------------------|
| Tinggi tampil | 400px (mobile) - 550px (desktop)         |
| Lebar tampil  | Full width                               |
| Rasio ideal   | **16:9**                                 |
| Rekomendasi   | **1920 x 1080 px**                       |
| Catatan       | Sisi kiri gambar tertutup overlay gelap; fokus visual di sisi kanan |

## Halaman Lainnya (Profil, Kontak, BKK, dll.)

Semua halaman menggunakan **banner hero** di bagian atas dengan format yang sama:

| Properti      | Nilai                    |
|---------------|--------------------------|
| Tinggi tampil | 400 - 500 px             |
| Lebar tampil  | Full width (100vw)       |
| Rasio ideal   | **16:5** atau **16:4**   |
| Rekomendasi   | **1920 x 600 px**        |
| Catatan       | Teks judul halaman muncul di atas gambar dengan overlay biru. Gunakan gambar yang tidak terlalu ramai. |
