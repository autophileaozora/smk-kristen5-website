## ğŸš¨ Common Issues & Solutions

### Issue 1: "Page not indexed"
**Causes:**
- Recent page, Google hasn't crawled yet
- robots.txt blocking
- noindex tag
- Low quality content

**Solution:**
1. Request indexing manually
2. Check robots.txt (should allow)
3. Remove noindex tag
4. Improve content quality
5. Add internal links to the page

### Issue 2: "Crawled - currently not indexed"
**Causes:**
- Low quality or thin content
- Duplicate content
- No unique value

**Solution:**
1. Add more content (500+ words)
2. Make content unique
3. Add images, videos
4. Improve page value

### Issue 3: "Discovered - currently not indexed"
**Causes:**
- Google found page but hasn't crawled yet
- Low priority

**Solution:**
1. Request indexing
2. Add sitemap
3. Improve internal linking
4. Wait (can take weeks)

### Issue 4: Low CTR despite good ranking
**Causes:**
- Boring title/description
- Not compelling
- Missing call-to-action

**Solution:**
1. Update meta title (add numbers, benefits, CTA)
2. Update meta description (add emotion, urgency)
3. Use schema markup
4. Add FAQ rich snippets
